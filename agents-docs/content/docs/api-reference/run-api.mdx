The **Run API** provides runtime endpoints for executing agent conversations and interactions.

## Key Features

- **OpenAI-Compatible** - Drop-in replacement for OpenAI's Chat Completions API
- **Streaming Support** - Real-time streaming responses with SSE and Vercel AI SDK protocols
- **A2A Protocol** - Agent-to-Agent communication following Google's specification
- **MCP Integration** - Model Context Protocol for enhanced agent capabilities

## Base URL

- **Development:** `http://localhost:3003`
- **Production:** Configure via `AGENTS_RUN_API_URL` environment variable
