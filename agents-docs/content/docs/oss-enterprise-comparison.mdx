## Overview

There are two options for how to build with Inkeep today:

- **Open Source (Self-hosted)**: Free, community-supported version with core framework features
- **Enterprise (Cloud or Self-hosted)**: Fully managed platform with out-of-box agents for common scenarios, advanced features, and dedicated support

With either option you can create agent-powered assistants, support bots, copilots, workflows, and more.

## Summary

The **Open Source** edition provides all the core features needed to build and deploy powerful AI agents with multi-agent architecture, MCP support, and visual building tools. It's perfect for developers and teams who want to start building agents in code or no-code.

The **Enterprise** edition adds managed services, pre-built agents, advanced analytics, enterprise-grade security, and dedicated support. It's ideal for organizations that want to accelerate their AI agent deployment with a trusted partner and managed platform.
