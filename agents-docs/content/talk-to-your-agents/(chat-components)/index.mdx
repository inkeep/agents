The Inkeep chat components provide pre-built UI elements to quickly integrate conversational AI into your application. These components handle the chat interface, message streaming, and user interactions out of the box.

## Available Components

| Component | Description |
|-----------|-------------|
| [Chat Button](/talk-to-your-agents/react/chat-button) | A floating button that opens a chat modal |
| [Custom Trigger](/talk-to-your-agents/react/custom-trigger) | Use your own button or element to trigger the chat |
| [Side Bar Chat](/talk-to-your-agents/react/side-bar-chat) | A chat panel that slides in from the side |
| [Embedded Chat](/talk-to-your-agents/react/embedded-chat) | A chat component embedded directly in your page |

## Framework Support

The chat components are available for:

- **[React](/talk-to-your-agents/react/chat-button)** - Native React components with full TypeScript support
- **[Next.js](/talk-to-your-agents/nextjs/chat-button)** - Same React components with App Router–friendly setup (dynamic import, `"use client"`)
- **[JavaScript](/talk-to-your-agents/javascript/chat-button)** - Vanilla JavaScript for any framework or plain HTML

## Setup

To use chat components, create an **app credential** — this configures domain restrictions, agent access, and end-user identity for your widget.

1. Open the Visual Builder Dashboard
2. Navigate to your Project → **Apps**
3. Click **Create App** and set the type to `web_client`
4. Add your **allowed domains** (e.g., `docs.example.com`)
5. Select a **default agent**
6. Copy your **App ID** (e.g., `app_a1b2c3d4e5f6`) — you'll pass this to the widget

See [App Credentials](/talk-to-your-agents/app-credentials) for the full auth flow details, security model, and API-based setup.

## Automatic Headers

The chat components automatically send the following headers with each request to provide client context:

| Header | Description | Example Value |
|--------|-------------|---------------|
| `x-inkeep-client-timestamp` | The client's current timestamp in ISO 8601 format | `2025-01-30T18:15:00.000Z` |
| `x-inkeep-client-timezone` | The client's timezone identifier | `America/New_York` |

These headers enable your agents to provide time-aware responses. See [Headers](/typescript-sdk/headers#widget-provided-headers) to learn how to use these values in your agent configuration.


## Custom Headers

You can pass custom headers to the chat components via the `headers` prop in `aiChatSettings`. In the following example, you'll pass the current page URL to the agent.

<Steps>
  <Step>
    #### Define the headers schema

    In the Agent Settings page, scroll down to **Headers schema** and add the following schema:

```json
{
  "type": "object",
  "properties": {
    "x-current-url": {
      "type": "string",
      "description": "The URL the user is currently viewing."
    }
  },
  "required": []
}
```

  </Step>
  <Step>
    #### Reference the header in your agent prompt

    Use the key defined in your headers schema with the `{{headers.*}}` template syntax:

```text
The user is currently viewing {{headers.x-current-url}}.
```

  </Step>
  <Step>
    #### Pass the header value from your chat component

<CodeGroup>
```tsx title="React"
import { InkeepChatButton } from '@inkeep/agents-ui'
import type { InkeepAIChatSettings } from '@inkeep/agents-ui/types'
import { useState, useEffect } from 'react'

export default function ChatButton() {
  const [currentUrl, setCurrentUrl] = useState<string>()

  useEffect(() => {
    const sync = () => setCurrentUrl(window.location.href)
    sync()
    window.addEventListener('hashchange', sync)
    window.addEventListener('popstate', sync)
    return () => {
      window.removeEventListener('hashchange', sync)
      window.removeEventListener('popstate', sync)
    }
  }, [])

  const aiChatSettings: InkeepAIChatSettings = {
    baseUrl: 'https://your-api.example.com',
    appId: 'YOUR_APP_ID',
    ...(currentUrl && {
      headers: {
        'x-current-url': currentUrl,
      },
    }),
  }

  return <InkeepChatButton aiChatSettings={aiChatSettings} />;
}
```

```js title="JavaScript"
const chatButton = Inkeep.ChatButton({
  aiChatSettings: {
    baseUrl: 'https://your-api.example.com',
    appId: 'YOUR_APP_ID',
    headers: {
      'x-current-url': window.location.href,
    },
  },
});

function syncUrl() {
  chatButton.update({
    aiChatSettings: {
      headers: { 'x-current-url': window.location.href },
    },
  });
}

window.addEventListener('hashchange', syncUrl);
window.addEventListener('popstate', syncUrl);
```
</CodeGroup>

  </Step>
</Steps>
