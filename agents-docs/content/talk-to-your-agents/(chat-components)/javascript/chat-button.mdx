The chat button component is a simple way to trigger the Inkeep chat bubble in your JavaScript application.

## Quick Start

<Steps>
  <Step>
    Add the below `<script>` tag to the `<head>` or `<body>` of your website.
    ```html
    <script
        type="module"
        src="https://cdn.jsdelivr.net/npm/@inkeep/agents-ui-js@0.15/dist/embed.js"
        defer
></script>
    ```
  </Step>
  <Step>
    Insert the Chat Button by using the `Inkeep.ChatButton()` function.

    ```ts
    const config: InkeepChatButtonProps = {
      aiChatSettings: {
        agentUrl: "https://your-api.example.com/run/api/chat",
        apiKey: process.env.INKEEP_API_KEY, // Your API key
      },
    };

    const chatButton = Inkeep.ChatButton(config);
    ```

  </Step>
</Steps>

## Custom Tool Component

Customize how tool calls are rendered in the chat by providing a custom `IkpTool` function via `aiChatSettings.components`.

The `IkpTool` function receives the following properties:
- **`tool`** - A `ToolUIPart` object containing the tool's name, arguments, state, and result
- **`approve`** - An async function to approve or reject tool calls that require user confirmation
- **`renderMarkdown`** - A utility function to render markdown text as a DOM node

### Example

```js
const config = {
  aiChatSettings: {
    agentUrl: "https://your-api.example.com/run/api/chat",
    apiKey: "YOUR_API_KEY",
    components: {
      IkpTool({ tool, approve, renderMarkdown }) {
        const container = document.createElement("div");

        // Handle tools that require approval
        if (tool.state === "requires-confirmation") {
          container.className = "tool-approval";
          container.innerHTML = `<p>Tool <strong>${tool.toolName}</strong> wants to execute.</p>`;

          const approveBtn = document.createElement("button");
          approveBtn.textContent = "Approve";
          approveBtn.addEventListener("click", () => approve(true));

          const rejectBtn = document.createElement("button");
          rejectBtn.textContent = "Reject";
          rejectBtn.addEventListener("click", () => approve(false));

          container.appendChild(approveBtn);
          container.appendChild(rejectBtn);
          return container;
        }

        // Handle completed tools
        if (tool.state === "result") {
          container.className = "tool-result";
          container.innerHTML = `<p>âœ“ ${tool.toolName} completed</p>`;
          return container;
        }

        // Handle tools in progress
        container.className = "tool-loading";
        container.innerHTML = `<p>Running ${tool.toolName}...</p>`;
        return container;
      },
    },
  },
};

const chatButton = Inkeep.ChatButton(config);
```

Return `null` from the function to hide specific tool calls from the UI.
